Hoe dit werkboek te gebruiken¶

Dit werkboek werd gemaakt met Jupyter Book, een open-source project om online interactieve boeken te ontwerpen waarvan de code ook effectief kan uitgerekend worden.

We leggen hier kort de concepten van interactieve boeken uit. De meeste features kunnen in de toolbar bovenaan de pagina gevonden worden.

Open Jupyter Notebook in de Cloud¶

Je kan de meeste pagina’s van dit boek openen in de cloud en van daarin de code live laten lopen. Gebruik hiervoor het raket icon bovenop de pagina klik op “Binder”om een versie van dezelfde pagina te open in de cloud.

Binder is een dienst om jupyter notebooks uit te voeren zonder voorafgaande configuratie of installatie. Merk wel op dat dat behoorlijk lang kan duren, dus moet je geduldig zijn.

Download het Jupyter Notebook¶

Je kan elke Jupyter notebook pagina van dit boek downloaden als een notebook file (.ipynb). Gebruik hiervoor het download icon en klik op “.ipynb”

Let op

Om te werken met deze .ipynb file, moet je Jupyter geinstalleerd hebben op je eigen computer.

Download als PDF¶

Je kan ook elke Jupyter notebook pagina van dit boek downloaden als een PDF file. Gebruik hiervoor het download icon en klik op “.pdf”

Werk full Screen¶

Om dit boek full screen te bekijken, klik op het icon bovenaan de pagina.

Meld een probleem op GitHub¶

Als je een probleem hebt of een suggestie, kan je dit melden op Github door naar het GitHub icon te gaan en klik op “open issue.”

Je kan me natuurlijk ook steeds emailen met een probleem of een suggesties op: jan.genoe@kuleuven.be

Klik om de gebruikte code te tonen¶

Om de leesbaarheid van dit boek te vergroten heb ik zo veel mogelijk de gebruikte code verborgen. Je kan deze steeds tonen door te klikken op “+” naast de verborgen code cell.

De bedoeling van het ter beschikking stellen van de gebruikte code is natuurlijk dat je zelf ook met deze code gaat experimenteren.

print("Hier is een voorbeeld van verborgen code. Klik op de '+' om deze te zien")
